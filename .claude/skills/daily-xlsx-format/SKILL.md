# daily_{Y}_{M}.xlsx の列構成

`create_daily.py` が ★営業日報{Y}年{M}月.xlsx（横持ち形式）から変換して出力するファイル。
`create_dashboard_img.py` の `load_data()` はこのファイルを読み込む前提になっているため、列構成を変更する場合は両方を揃えて直すこと。

## 日次データシート（テーブル形式、ヘッダーは行1）

| 列 | 内容 |
|---|---|
| 日付 | 日付（yyyy/mm/dd） |
| 曜日 | 月〜日 |
| 定休 | 休日は "休" |
| 祝日 | 祝日フラグ |
| 純売上高 | 税抜売上 |
| 消費税 | 消費税額 |
| 総売上高 | 税込売上（純売上高＋消費税） |
| 現金〜夕食売上 | 支払方法・カテゴリ別・客数・食事別売上 |

## 商品別シート（テーブル形式）

FOOD / DRINK の商品名・単価・数量・金額（日別・順位別）

## 元データ（★営業日報{Y}年{M}月.xlsx）側の行番号（`create_daily.py` 内の固定レイアウト）

「データ」シートの行番号は `create_daily.py` 冒頭の定数（`R_TEIKYU`, `R_JUN`, `R_TOTAL` など）で定義されている。元Excelのレイアウトが変わった場合はここを修正する。
